# Лабораторна робота №4 - Цикл подій (event loop)

### Варіант завдання: 3

- `reverse <arg>`

  Інструкція “перевертає” вхідний рядок.

  Приклад виклику у вхідному файлі:

  `reverse hello`

  Очікуваний результат:

  `olleh`

